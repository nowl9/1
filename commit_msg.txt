feat(execution): deterministic PM benchmark settlement (paper-trade step 3)

Build step 3 of docs/paper-trading-plan.md (sections 3.3, 4.2; Fork 2).
Polymarket paper positions now settle at expiry against a DETERMINISTIC
CF-Benchmarks-RTI-style terminal-digital benchmark MODEL -- NOT
Polymarket's live oracle -- so a replay (build step 5, separate follow-up)
can reproduce settlement from the recorded surface with no live call.

- New btc_pm_arb/execution/benchmark_settlement.py:
    * benchmark_settlement_price(benchmark_price, strike, direction) -- a
      pure, total terminal-digital model returning the YES settlement price
      in {0,1}: above-strike -> 1.0 (>=), below-strike -> 1.0 (<).
    * PaperBenchmarkSettler.settle_due() -- the deterministic, no-HTTP
      counterpart to KalshiSettlementPoller.  PM-only; expiry-gated through
      the injected sim-clock; fail-safe-open skips (non-PM, unexpired,
      missing strike, missing fixing) with logs, never a guessed settlement.

- paper_settlement.py: extracted build_settlement_record() + the
  win/loss/push classifier as module-level commons; KalshiSettlementPoller
  now uses them too (record + log byte-identical) -- "only the price SOURCE
  differs" made literal.

- paper_ledger.py: PaperOrderRecord gains optional strike + direction
  (defaulted, schema_version 1 unchanged) so the settler can evaluate the
  threshold predicate without an oracle.

- main.py: order records capture strike/direction; Agent builds a
  PaperBenchmarkSettler with a deterministic benchmark_price_fn
  (_benchmark_btc_price -> latest observed Deribit index) and drives
  settle_due() once per scan tick.  No-op until a position reaches expiry,
  so short live runs are unaffected.

Tests: tests/test_execution/test_benchmark_settlement.py (11) -- model
above/below/boundary; settler writes a PaperSettlementRecord and closes the
position; DETERMINISM (same surface -> same settlement_price + P&L twice);
idempotent re-pass; skips non-PM / unexpired / missing-strike /
missing-fixing.  Full suite 712 passed (701 prior + 11).
